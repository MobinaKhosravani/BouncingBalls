# **Bouncing Ball Game (C++ & SDL2)**

A simple bouncing ball game built using **C++** and **SDL2**, demonstrating basic game loops, rendering, event handling, and simple physics.  
This project was created as part of a **basic programming course at Sharif University of Technology**, and helped me practice game development fundamentals and SDL-based graphics programming.
